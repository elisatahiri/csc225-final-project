About This Project

Project Overview

This web application displays characters from the Harry Potter universe using the HP-API. Users can browse all characters, filter by house, search by name, and view detailed information about each character.

API Information

API Used: HP-API

Base URL: https://hp-api.onrender.com

Authentication: None required

Endpoints Used:

  • /api/characters - Get all characters
  • /api/character/{id} - Get character by ID
Technologies Used
  • HTML5
  • CSS3
  • Bootstrap 5.3
  • Vanilla JavaScript (ES6+)
  • Fetch API
Features
  • Browse all Harry Potter characters
  • Filter characters by Hogwarts house
  • Search characters by name
  • View detailed character information
  • Responsive design for all devices
How to Use
  1. On the home page, browse through the character cards
  2. Use the search bar to find specific characters
  3. Use the dropdown to filter by house
  4. Click on a character card to view more details
